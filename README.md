# 👻 ФАЗМОФОБИЯ — Web

Клон Phasmophobia для мобильного браузера: top-down 2D хоррор с «туманом войны»,
охотами и полноценным циклом паранормального расследования.

**Без фреймворков, без сборки, без единого ассета** — чистый JavaScript
(ES-модули), Canvas 2D и WebAudio. Вся графика рисуется кодом, весь звук
синтезируется на лету.

## Запуск

Любой статический сервер из корня репозитория:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```

Или включите GitHub Pages (Settings → Pages → Deploy from branch) — игра
работает как есть. Играть лучше на телефоне в ландшафтной ориентации.

## Управление

| Мобильный | ПК |
|---|---|
| Левая половина экрана — джойстик движения | WASD / стрелки |
| Правая половина — направление взгляда/фонарика | Мышь |
| ✋ — контекстное действие (двери, свет, укрытия, фургон) | E |
| ⚙ — использовать/разместить предмет | R |
| 🔁 — смена слота, 🔦 — фонарик, 📓 — журнал | Q, F, J |

## Игра

- **Дом**: 9 комнат + подвал из 3 комнат, двор с фургоном-штабом.
- **12 призраков** (Дух, Мираж, Фантом, Полтергейст, Банши, Джинн, Мара,
  Ревенант, Тень, Демон, Юрэй, Онрё) — у каждого уникальная тройка улик и
  особое поведение.
- **7 улик**: ЭМП-5, спиритбокс (спрашивайте в темноте!), УФ-отпечатки,
  призрачные огни (только через видеокамеру на мониторе фургона), запись в
  книге, минусовая температура, DOTS-силуэт.
- **Рассудок** тает в темноте; чем ниже — тем чаще охоты. Прячьтесь в шкафах,
  жгите благовония, ставьте распятия. Смерть = скример.
- В фургоне: 3 слота снаряжения из 11 предметов, ночной монитор камер,
  датчик активности, средний рассудок.
- Награды, статистика и прогресс сохраняются в localStorage.

## Технологии

- Туман войны: raycast-полигон видимости по стенам/закрытым дверям/шкафам.
- Свет: offscreen-канвас тьмы в половинном разрешении, источники «выбиваются»
  через `destination-out` внутри полигона видимости.
- Атмосфера: динамическая виньетка (пульсирует от рассудка), киношное зерно,
  частицы (пыль в луче, пар дыхания в холоде, шлейф призрака).
- ИИ призрака: A* по тайлам, стейт-машина (блуждание → взаимодействия →
  манифестации → охота), слух и прямая видимость при преследовании.
- Звук: полностью процедурный WebAudio — дрон, ветер, шаги, скрипы,
  сердцебиение, помехи спиритбокса, «голос» призрака, стинг охоты.
