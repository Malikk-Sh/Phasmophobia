# Vetted audio sources

These sources were selected for the optional realistic audio pack because they fit the game and have redistribution-friendly licenses.

## Primary CC0 packs

- **100 CC0 SFX #2** by rubberduck — OpenGameArt page: <https://opengameart.org/content/100-cc0-sfx-2>. The page marks the license as **CC0** and lists useful categories for this game: air flow, doors, footsteps, ambient loops, switches, thunder, metal and wood hits.
- **100 CC0 metal and wood SFX** by rubberduck — OpenGameArt page: <https://opengameart.org/content/100-cc0-metal-and-wood-sfx>. The page marks the license as **CC0** and includes metal/wood doors, slams, cracking and squeaking wood.
- **Dark Ambiences** by Ogrebane — OpenGameArt page: <https://opengameart.org/content/dark-ambiences>. The page marks the license as **CC0** and includes five dark ambience WAV files suitable for room tone and basement ambience.

## Optional attribution pack

- **Horror Sound Effects Library** by Little Robot Sound Factory — OpenGameArt page: <https://opengameart.org/content/horror-sound-effects-library>. This pack is useful for horror ambiences, breathing, monsters and screams, but it is **CC-BY 3.0**, so it is not included in the automatic CC0 downloader. Use it only if you are comfortable preserving attribution in-game or in documentation.
